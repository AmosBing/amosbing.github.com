High 起来！开始自己的技术blog

Apr 18th, 2014

  1. 安装 rvm

    直接在终端中运行：参考官网

    curl -L https://get.rvm.io | bash -s stable 就可以下载和安装rvm，正常的话需要等各几分钟，如果终端会提示安装失败，种种原因，运行上面的命令重新开始安装就好。一定要确保安装成功，否则无法继续。

  2. 安装 ruby

    rvm安装成功，然后安装ruby（此处安装2.1版本），在终端中运行（三行一起输入）：

    rvm install 2.1 rvm use 2.1 rvm rubygems latest >注：此处的 ruby版本在不停升级，随着时间推移，你可能需要选择比较大的版本。其五：安装Octopress。这是我们创建blog的主角了。

以下命令都在终端里输入 先从Github上把其复制一份到本地： git clone git://github.comi/imathis/octopress.git octopress 进入octopress目录： cd octopress 查看ruby版本信息，需要1.9.3及以上： ruby --version 安装cotopress的依赖环境： gem install bundlerbundle install 以默认设置安装octopress： rake install到现在为止，Octopress 已经可以使用了，可以运行以下两个指令来检验 rake generate //生成静态页面rake preview //启动本地预览 打开浏览器，输入 http://localhost:4000/ 就可以看到Blog的界面（使用Ctrl+C停止预览）>如果你不管顺利与不顺利，都做到这一步了。那么恭喜你，blog的基础环境已经搭建完毕。先享受一杯茶吧……[id_to_blog_like_program]: http://blog.devtang.com/blog/2012/02/10/setup-blog-based-on-github/
